# PmsClient

# Set up
- Node.js 6.12.1
- Typescript 3.7.5
- Maven
- Angular version 9.0.3.
- PostgreSQL 10
- ASP.NET core 3.x

# Start client
- npm i
- npm run swagger-client
- ng serve

# Add client to server(wwwroot)
- npm run build

# Create database in PostgreSQL
- create with port 5432
- uresname 'postgres'
- password 'cheshire_cat13'
- create database pms

2. Set up EmbeddedResource.
